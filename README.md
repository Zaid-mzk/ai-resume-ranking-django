# AI Resume Ranking Web Application

## 📌 Description
This project is an AI-based multimodal resume ranking web application developed using Python and Django. It is designed to assist in large-scale recruitment by analyzing and ranking resumes based on predefined criteria.

The system allows users to upload resumes, processes them using backend logic, and generates ranked results to help identify the most suitable candidates efficiently.

---

## 🚀 Features
- Resume upload functionality
- AI-based resume ranking
- User and Admin modules
- Resume analysis and prediction
- Clean and responsive web interface

---

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- SQLite (default database)

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/Zaid-mzk/ai-resume-ranking-django.git

2. Navigate to project folder:
   cd ai-resume-ranking-django

3. Run the server:
   python manage.py runserver

4. Open in browser:
   http://127.0.0.1:8000/
